# Proposal: add-chatter-crm

## Why

Чатеры ведут переписку с фанами от лица контент-моделей, держа несколько активных диалогов одновременно, а тимлид должен в реальном времени видеть нагрузку чатеров и диалоги, где фан ждёт ответа слишком долго. Сейчас системы нет — это greenfield-реализация тестового задания (см. `Tech-Task-Backend+Frontend-Django_Vue3.md`).

## What Changes

- Новый backend: Django + DRF (REST для истории и списков) + Channels (WebSocket для live-событий), PostgreSQL, Redis.
- Новый frontend: Vue 3 (Composition API) + Pinia — рабочее место чатера и монитор тимлида.
- Рабочее место чатера: список диалогов (последнее сообщение, время, счётчик непрочитанных), окно переписки с пагинацией истории, отправка обычных и PPV-сообщений (с ценой).
- Монитор тимлида: чатеры онлайн, число диалогов в работе, число просроченных ожиданий (> X минут), обновление в реальном времени.
- Метрика времени ответа: таймер запускается с сообщения фана, останавливается с ответа чатера; просрочки подсвечиваются.
- Надёжность: переподключение WebSocket с догрузкой пропущенных сообщений без дублей и потерь.
- Demo-инфраструктура: docker-compose (одна команда), сидер тестовых данных, эмуляция входящих сообщений фана, деплой на Railway.

## Capabilities

### New Capabilities

- `auth`: сессионная авторизация, роли chatter / teamlead, доступ к интерфейсам по роли.
- `chat`: диалоги и сообщения — список диалогов чатера, история с cursor-пагинацией, отправка обычных и PPV-сообщений, счётчик непрочитанных.
- `realtime`: WebSocket-канал — доставка live-событий, Channels-группы, переподключение и resync без дублей и потерь.
- `teamlead-monitor`: presence чатеров (grace period), метрика времени ответа и просрочек (порог X), live-дашборд тимлида.
- `demo-tooling`: запуск одной командой, сидер, эмуляция фана, конфигурация порогов через env, деплой.

### Modified Capabilities

Нет — проект новый, существующих спек нет.

## Impact

- Новые репозитории: `justfans-backend` (Django, docker-compose), `justfans-frontend` (Vue 3 SPA).
- Зависимости: Django, DRF, Channels, channels-redis, daphne/uvicorn, psycopg, pytest; Vue 3, Pinia, Vue Router, Vitest.
- Инфраструктура: PostgreSQL, Redis, nginx (статика фронта + прокси `/api` и `/ws`), Railway (деплой).
